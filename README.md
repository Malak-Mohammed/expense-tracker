# Expense Tracker

A secure expense tracking platform built with Spring Boot, Hibernate, and MySQL.

## Features
- User authentication with JWT
- Expense CRUD operations
- Category-based reporting
- CSV/PDF export
- Dockerized deployment

## Tech Stack
- Java 17
- Spring Boot
- Hibernate / JPA
- MySQL
- Docker

## Getting Started
1. Clone the repo
2. Configure `application.yml` with your MySQL credentials
3. Run `mvn spring-boot:run`
